מה שלומך? 
הכנתי לך דף הנחיות לאיך להתמודד עם הפרוייקט שלי. 

אז נתחיל במבנה.
הפרוייקט מורכב מ 3 אפליקציות. 
C# - מרכז כל הלוגיקה.
NodeJS - הסימולטור שמניע ושולח מטוסים ללוגיקה. 
React - הממשק הגרפי של התוכנית

אז לפני שאפשר להריץ את הפקוייקט המלא צריך להריץ  "npm i" בטרמינל של התקייה של הNODE ובנוסף להריץ "npm i" גם בטרמינל של התקייה של ה REACT.

אחרי שזה מתבצע אפשר להריץ את כל אחד מהם בניפרד. את ה Node & React אפשר להריץ בעזרת "npm start" וה C# דרך הVS. אבל גם דאגתי להוסיף קטע קוד ב C# שדואג להריץ את כל השלושה, ואפילו להדליק את הדפדפן. אבל בשביל שזה יקרה צריך להשאיר את הסדר של התיקיות כמו שאני שולח:  AirPort + sulution, airport-ui, node simulator כולם אחד ליד השני באותה תיקיה. 
בקיצור לבצע UnZip בדיוק באותו מקום כמו שאני שולח.
